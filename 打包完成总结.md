1. **重新打包**
   ```powershell
   python -m PyInstaller --onefile --windowed --name GlotSub main.py
   .\create_installer.bat
   ```

2. **测试程序**
   ```powershell
   .\test_exe.bat
   ```

3. **创建安装程序**（使用 Inno Setup）
   - 打开 `setup.iss` 文件
   - 在 Inno Setup 中编译

### 对于用户

1. **解压安装包**
   - 解压 `GlotSub_安装包.zip` 到任意目录

2. **安装 Tesseract OCR**（必需）
   - 下载: https://github.com/UB-Mannheim/tesseract/wiki
   - 安装时勾选中文语言包
   - 添加到系统 PATH

3. **运行程序**
   - 双击 `GlotSub.exe` 启动
