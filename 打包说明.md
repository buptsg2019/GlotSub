# GlotSub 打包说明

## 打包结果

打包已完成！生成的文件位于 `dist` 目录：
- **GlotSub.exe** - 主程序可执行文件（约67MB）

## 安装包结构

建议的安装包结构：

```
GlotSub_安装包/
├── GlotSub.exe          # 主程序
├── README.md            # 使用说明
├── INSTALL.md           # 安装指南
└── 安装说明.txt         # 快速安装说明
```

## 用户安装步骤

### 前置要求

1. **安装 Tesseract OCR**（必需）
   - 下载地址：https://github.com/UB-Mannheim/tesseract/wiki
   - 安装时务必勾选中文语言包（Chinese Simplified）
   - 将安装目录添加到系统 PATH

2. **Windows 系统要求**
   - Windows 7/10/11
   - 无需安装 Python（已打包在exe中）

### 安装步骤

1. 将 `GlotSub.exe` 复制到任意目录（建议创建专用文件夹）
2. 双击运行 `GlotSub.exe`
3. 如果提示缺少 Tesseract OCR，请先安装 Tesseract OCR

## 测试打包结果

运行以下命令测试可执行文件：

```powershell
cd dist
.\GlotSub.exe
```

## 创建安装包

### 方法1：直接分发（最简单）

直接将 `dist/GlotSub.exe` 和说明文档打包成 ZIP 文件分发。

### 方法2：使用 Inno Setup 创建安装程序（推荐）

1. 下载安装 Inno Setup：https://jrsoftware.org/isdl.php
2. 使用提供的 `setup.iss` 脚本创建安装程序

### 方法3：使用 NSIS 创建安装程序

1. 下载安装 NSIS：https://nsis.sourceforge.io/Download
2. 创建安装脚本

## 注意事项

1. **Tesseract OCR 依赖**：程序需要系统安装 Tesseract OCR，无法打包进exe
2. **首次运行**：首次运行可能需要几秒钟解压
3. **杀毒软件**：某些杀毒软件可能误报，这是正常现象（PyInstaller打包的常见问题）
4. **文件大小**：exe文件约67MB，包含了所有Python依赖

## 优化建议

如需减小文件大小，可以：
1. 使用 `--exclude-module` 排除不需要的模块
2. 使用 UPX 压缩（已启用）
3. 考虑使用虚拟环境打包
